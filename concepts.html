Devon’s Key Concepts

  1. Tim Berners-Lee – Creator of the first version of HTML in the early 1990s.
  2. HTML 1.0 – The first version used for basic web pages with text and links.
  3. HTML 2.0 – Early standardization that introduced forms and better structure.
  4. HTML 4.01 – A major update that supported CSS and improved layout design.
  5. XHTML – A stricter XML-based version of HTML used for improved consistency.
  6. HTML5 – The modern version with support for multimedia, APIs, and semantics.
  7. W3C – The standards organization responsible for HTML’s evolution.
